Definitions

Vacant and underutilized rule

Vacant means county IMPRVT_VAL is 0. Likely underutilized means county IMPRVT_VAL is greater than 0 but less than or equal to 20% of county LAND_VAL. Active means improvement value is above that threshold.

Method

Opportunity score and clustering

Score weights vacancy or likely underutilization, lot size, low improvement value, lower assessed value, civic/nonprofit ownership, and QOZ status. Clusters group high-score parcels by nearby centroid grid cells, then size markers by parcel count.

Dataset

Structured parcel fields

Each parcel includes block/lot, address, inferred ownership, land-use classification, vacancy status, assessed value, lot size, zoning, inferred geography, centroid, and polygon geometry.

Operations Manual

How to Use the Newark City Parcel Dashboard

This guide explains the dashboard workflow, parcel classification logic, map operations, selection tools, and export options.

1. Start With Filters

Use the status, ownership, geography, zoning, and search controls to narrow the visible parcel universe. Filters update KPIs, charts, parcel lists, clusters, and exports.

2. Read Utilization Status

Vacant means county IMPRVT_VAL is 0. Underutilized means IMPRVT_VAL is greater than 0 and less than or equal to 20% of LAND_VAL. Active parcels are above that ratio.

3. Use the Map

Click a parcel polygon to open the selected parcel view. Use the full record link to inspect every available field. Toggle clusters with the C control.

4. Select Parcels

Use the checkbox beside high-opportunity parcels or the popup select button. Selected parcels can be exported separately from the current filtered dataset.

5. Export Data

Export all visible parcels, selected parcels, a JSON data package, or an HTML report. The report summarizes KPIs, top opportunity parcels, and current filter conditions.

6. Interpret Ownership

Ownership type is inferred from REGRID_owner and REGRID_lbcs_ownership_desc. Use owner subtype and confidence in the full parcel record before making decisions.
